# Static Website Hosting using AWS S3

## Overview
This project demonstrates how to host a static website using Amazon S3.  
The website is publicly accessible through the S3 static website hosting feature.

## Project Description
I created a static website using HTML and CSS and deployed it on an Amazon S3 bucket by enabling static website hosting.  
The website can be accessed using the S3 website endpoint.

## Technologies Used
- HTML
- CSS
- Amazon S3

## AWS Services
- Amazon S3 (Static Website Hosting)

## Live Website
http://<your-s3-website-endpoint>

## Steps Implemented
1. Created an S3 bucket
2. Enabled static website hosting
3. Uploaded HTML and CSS files
4. Configured public access using bucket policy
5. Verified website access using S3 website endpoint

## What This Project Demonstrates
- Understanding of object storage concepts
- Hosting static content on cloud infrastructure
- Basic AWS S3 configuration
- Public access management using bucket policies
